Il progetto

I dati di Patrimoni trasparenti sono disponibili in diversi formati aperti.

  • Per data journalists e data analysts: download dei dati riassuntivi in formato CSV [link a pacchetto zip con 2 files CSV (uno per anno) e un file CSV di metadati] I nomi dei file sono: per i csv "openpolis-patrimoni-trasparenti-anno-YYYY.csv" mentre lo zip si chiama "openpolis-patrimoni-trasparenti.zip" I campi dei csv sono descritti qui: #23 da dove è possibile scaricare anche il file dei metadati pronto che si deve chiamare "openpolis-patrimoni-trasparenti-tracciato-record-metadati.csv"
  • Per programmatori: download di tutti i dati in formato JSON
  • Per programmatori avanzati: accedi alle nostre Api
Tutti i dati sono di proprietà dell'Associazione Openpolis, che li rilascia con licenza CC BY-SA 3.0.
